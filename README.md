# RRL-App-Server
Server for RoyalRoad-App

# Running on:
Ubuntu 18.04 LTS

Dynamic Dns w/ afraid.org(royalroad.chickenkiller.com)

Python 3

# To Use
1. Install python3
2. Run:
   - `pip3 install requests bs4 ebooklib Click`
   - `git clone https://github.com/LaineOtto/RRL-App-Server/`
   - `cd RRL-App-Server`
   - `python3 getEbook.py --help`
3. Read help page

# Dependencies
requests

BeautifulSoup

ebooklib

click
